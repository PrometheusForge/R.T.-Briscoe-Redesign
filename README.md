# RT Briscoe Nigeria Plc – Corporate Homepage

A fully responsive, single-file HTML/CSS/JS website for RT Briscoe Nigeria Plc, a diversified conglomerate with automotive, industrial equipment, real estate, and air compressor divisions.

## ✨ Features

- **12 strategic sections** – Hero → Trust bar → Divisions → Stats → Testimonials → Carousel → Quote form → etc.
- **Sliding carousel** – Multi‑content slider with dot indicators showing current position.
- **Two color variants**:
  - *Premium*: Deep navy (`#0B1929`) + gold (`#C9A84C`)
  - *Brand original*: `#383e77` + `#2a2a2a` footer, original logo
- **Lead capture form** – Client‑side validation (name, phone, email, business unit, subject, message). Integrates with Formspree.
- **Mobile‑first responsive** – Breakpoints at 480px, 768px, and 1024px+.
- **Accessibility** – Semantic HTML, ARIA labels, keyboard navigation, WCAG contrast.
- **No dependencies** – Vanilla JS, inline CSS, self‑contained HTML file.

## 🖼️ Preview

> *Open `index.html` in any modern browser. Switch color variants by replacing the CSS section.*

## 🛠️ Tech Stack

- HTML5 (semantic markup)
- CSS3 (Grid, Flexbox, custom properties, media queries)
- Vanilla JavaScript (form validation, carousel logic, smooth scroll)
- Google Fonts (Playfair Display + Poppins)
